    # cnpm可能会在mac底下安装报错

    大多是没有获取管理员权限的缘故，此时可以在终端命令前添加sudo

    sudo npm install -g cnpm --registry=https://registry.npm.taobao.org 
如果还是显示权限问题则需要获取最高权限
sudo npm install -g cnpm --registry=https://registry.npm.taobao.org --verbose
此时会提示输入开机密码，输入基本就能完成安装
执行cnpm -v查看安装成功

    # vscode 自动编译ts

    在终端或者命令行中定位到自己到目录执行

    tsc --init
    1

    运行结束会生成一个tsconfig.json的配置文件， 文件内容都是一些编译配置如outDir等相对输出目录都可以自行调整， 在vscode中菜单都终端栏目下配置任务选择typescript监视任务， 之后运行这个已配置都任务即代码窗口底下都终端会显示一个运行都tsc监视任务即自动编译监视成功。 此时修改ts文件后保存会自动编译生成js文件。
